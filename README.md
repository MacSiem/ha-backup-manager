# HA Backup Manager

![HA Backup Manager Screenshot](screenshot.png)

Advanced backup management for Home Assistant. Schedule automated backups, manage retention policies, and monitor backup health.

## Installation

### HACS (recommended)

1. Open HACS in Home Assistant
2. Go to **Frontend** section
3. Click the three dots menu > **Custom repositories**
4. Add `https://github.com/MacSiem/ha-backup-manager` as **Dashboard**
5. Install **HA Backup Manager**
6. Restart Home Assistant

### Manual

Copy the contents to `/config/www/community/ha-backup-manager/`
